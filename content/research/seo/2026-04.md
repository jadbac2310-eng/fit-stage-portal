# キーワード調査レポート

---

## 1. 調査方針

本調査はサービスページ（LP）のキーワード設計を目的とする。ブログ記事とは異なり、LPは「検討→申込み」の導線を担うページであるため、以下の方針でキーワードを選定した。

- **コンバージョン意図のあるキーワードを優先**（「とは」「料金」「メリット」「エリア」系）
- **検索数よりSEO難易度の低さを重視**（大手と正面衝突せず、取れる場所から取る）
- **1ページに集中**（エリア別・ターゲット別に複数LP展開するほど各サブKWの検索数がない）

---

## 2. 調査フロー

```
Phase 1: キーワード探索（4 API × 4起点KW = 16リクエスト）
  ├── /v1/suggest-keywords   サジェストKW + メトリクス
  ├── /v1/related-keywords   関連KW + メトリクス
  ├── /v1/other-keywords     LSI/PAA（潜在KW・質問）
  └── /v1/question-search    よくある質問

Phase 2: 抜け漏れ確認（7リクエスト）
  ├── /v1/ranking-keywords   メインKWで上位表示ページが同時ランクインするKW
  └── /v1/influx-keywords    競合4サイトが実際に流入を獲得しているKW
```

## 8. 競合との差別化ポイント（ページ設計に反映すべき訴求）

Dr.トレーニング（最強競合）は全国展開の大手チェーン。FIT STAGEが訴求すべき差別化は以下。

- **関西ローカル特化** — 大阪・兵庫・京都に絞った密度の高いエリア対応
- **累計15,000セッション** — 実績の具体的な数字で信頼感を担保
- **ホームジム設営とのセット** — 大手にはない「環境構築＋指導」のオールインワン提案
- **LINE完結** — 問い合わせのハードルを下げるLINE一本での導線
